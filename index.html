Pasteur4oa Data Visualisations

Introduction

This area is being used to store data visualisations created for the PASTEUR4OA Project.

Questions that can be asked by data visualisations: https://pad.okfn.org/p/pasteur4oa-data-viz - Please do add any other ideas

ROARMAP data

All data available from the Registry of Open Access Repository Mandates and Policies (ROARMAP).

  • Number of Open Access Policies Worldwide
  • Number of Open Access Policies Worldwide - map
  • Number of Open Access Policies that are mandatory
  • When Open Access Policies require deposit
  • Open Access Policies that mention APCs
  • Google spreadsheet map with live data

Other data

Bibliographic metadata indexed by the Thompson-Reuters database (WoK).

  • Number of Deposits in Institutional Repositories

Publication date was estimated based on the Altmetrics database and WoK publication date.

  • Deposit latency on Open Access and restricted access publications
